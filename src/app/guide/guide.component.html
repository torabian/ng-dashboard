Angular 8 Dashboard

ng-dashboard is a modular dashboard based on Angular. You can create your own dashboard, based on tools provided on this library 

    npm i @ng-dashboard/core --save

@ng-dashboard/core comes with the authentication as well.

Teams Sections

In many applications, users can have teams. It's useful when you want to access the same data within a group of people, or having multiple teams. This feature is beta. At the moment you can access teams data using ConfigurationService
